# Mebel Store - Oshxona Modullari Savdo Platformasi

## 📋 Loyiha Haqida

Telegram Bot + WebApp orqali oshxona mebellari savdo platformasi.

## 🛠️ Texnologiyalar

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Backend:** Python (Telegram Bot)
- **Database:** Supabase (PostgreSQL)
- **Hosting:** GitHub Pages
- **Auth:** Supabase Auth (Telegram ID + Email)

## 📁 Strukturasi

```
Mebel_store/
├── webapp/
│   ├── index.html      # Landing page
│   ├── app.html        # Telegram WebApp
│   ├── admin.html      # Admin panel
│   └── assets/
├── supabase/
│   └── schema.sql      # Database schema
├── telegram/
│   └── bot.py          # Telegram bot
└── .github/
    └── workflows/
        └── deploy.yml  # CI/CD
```

## 🚀 Ishga Tushirish

1. `.env.example` → `.env` nusxa oling
2. Supabase'da `schema.sql` ishga tushiring
3. GitHub Secrets ga kalitlarni qo'shing
4. GitHub Pages'ni yoqing
